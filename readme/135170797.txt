This is the server for Socialgorithm's self-driving car simulator, featuring socket listeners that allow clients to connect, receive sensor data and send control data.
Make sure you have installed at least go 1.11, as that's the version that introduced support for go modules. Dependencies for the server will be automatically installed when you start it for the first time.
