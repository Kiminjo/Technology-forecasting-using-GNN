Room number recognition service for indoor self-driving cars. This service analyzes all contours in every single frame, finds the Room Number Plate, and extracts the number from the plate.
.
SVHN Model is a CNN model based on YOLOv5-small model, which is trained with SVHN Dataset(format 2).
For more information, visit this repository: icns-distributed-cloud/YOLOv5-SVHN
